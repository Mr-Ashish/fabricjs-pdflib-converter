# Examples

This directory contains example scripts demonstrating various features of canvas-pdflib-converter.

## Running Examples

```bash
# Build the project first
npm run build

# Run an example
npx ts-node examples/basic.ts
```

## Available Examples

### `basic.ts`
Demonstrates basic shapes: rectangles, circles, ellipses, triangles, and lines.

### `text-rendering.ts`
Demonstrates text rendering with different fonts, sizes, alignments, and multi-line text.

### `images.ts`
Shows how to work with images (placeholder example - add your own image data URLs).

### `vector-paths.ts`
Demonstrates SVG paths, polylines, polygons, and dash patterns.

## Output

Each example creates a PDF file in this directory (e.g., `output-basic.pdf`).
