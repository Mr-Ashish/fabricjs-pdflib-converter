# Canvas to PDF Converter - Demo App

An interactive demo application for testing the canvas-pdflib-converter library.

## Features

- **Interactive Canvas**: Full Fabric.js canvas with drag, resize, rotate
- **Object Creation**: Add rectangles, circles, triangles, text, and lines
- **Property Editor**: Edit position, colors, stroke, opacity, text properties
- **Canvas Settings**: Change background color and dimensions
- **PDF Export**: One-click export to PDF using the library

## Running the Demo

```bash
# From the demo directory
npm install
npm run dev
```

The app will open at http://localhost:5173

## Usage

1. **Add Objects**: Click the tool buttons in the sidebar to add shapes or text
2. **Edit Properties**: Select an object to see its properties in the sidebar
3. **Modify Canvas**: Change background color and canvas dimensions
4. **Export**: Click "Export to PDF" to download the generated PDF

## Testing the Library

This demo is useful for:
- Visual verification of exported PDFs
- Testing different object types and properties
- Debugging rendering issues
- Demonstrating library capabilities
